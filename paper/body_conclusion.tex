\section{Conclusion}

We introduced \textbf{Baihu}, a billion-frame, fully human-teleoperated, multi-platform robot manipulation dataset for embodied foundation model pretraining and evaluation. Baihu contains \textbf{9521.75 hours} of robot data, \textbf{2989 tasks}, \textbf{513575 episodes}, and \textbf{1028349814 frames} across \textbf{13 robot platforms}. The dataset converts platform-specific HDF5 robot operation records into the LeRobot v2.1 format, providing a standardized data foundation for training vision-language-action models and generalist robot policies.

Baihu is not only large in scale, but also diverse across robot platforms, task types, objects, and temporal horizons. Normalized ALL MSE is reported as the primary evaluation metric because it evaluates the full action vector, while Normalized Joint MSE is reported as an auxiliary diagnostic metric for joint-related action prediction. The dataset supports pretraining that benefits cross-platform action prediction.

We further evaluated the utility of Baihu through offline open-loop zero-shot evaluation on external datasets excluded from Baihu training. Across 13 platforms and 42 paired task-dataset records, a checkpoint trained on Baihu for one epoch reduced \textbf{Normalized ALL MSE by 96.79\%} and \textbf{Normalized Joint MSE by 99.42\%} compared with a no-training checkpoint. Each task-level result was averaged over five evaluation trajectories. These results suggest that Baihu provides effective large-scale supervision for robot action prediction and can serve as practical infrastructure for future embodied foundation model development.

Future work will extend Baihu with additional benchmark protocols, more baseline models, closed-loop evaluations, data-scaling studies, and clearer release documentation. Together, these directions can further strengthen Baihu as a reusable data foundation for large-scale embodied intelligence research.
